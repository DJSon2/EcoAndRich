## 개발 환경
> * Language : Java(SpringFramework 사용)     
> * DB : MySql      
> * IDE : Spring Tools 4

## 추가한 Starter 목록
> * JDBC API
> * Spring Boot DevTools
> * Spring Web
> * Spring Data JPA
> * Spring Data Jpa
> * MySQL Driver
## 그 외에 추가한 dependency
> * Model Mapper
> * log4jdbc(로깅 처리를 위한)
> * open api
> * swagger 관련
> * json 및 XML Parser
